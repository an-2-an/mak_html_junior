Создать в одной папке :
1) страница index.html
2) папку css с файлом style.css
3) папку img для картинок

На странице index.html:
- подключить style.css
- подключить font awesome
- сделать фоновый рисунок
- создать несколько div-ов с использованием
	значков font awesome,
	изменения внешнего вида при наведении курсора
